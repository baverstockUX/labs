# IQ Labs Innovation Portal

A high-tech innovation showcase platform built for OneAdvanced | IQ Labs. This accessible portal demonstrates the "Vibrant Alphas" and experimental apps currently in development.

## Features

- **High-Tech Aesthetic:** Dark mode interface with "Big Stone" background and neon accents (Tangerine, Coral Red, Electric Purple).
- **Interactive Hero:** 
  - Vertical light curtain effect with noise texture.
  - Interactive constellation animation that reacts to mouse movement (`ConnectedPointsBackground`).
- **Project Showcase:** Glassmorphism cards displaying prototype status (Alpha, Beta, Prototype).
- **Parallax Scrolling:** Immersive background color shifts powered by Framer Motion.
- **"Pitch an Idea":** Integrated modal for submitting new innovation concepts.

## Tech Stack

- **Framework:** Next.js 16 (Turbopack)
- **Styling:** Tailwind CSS v4
- **Animation:** Framer Motion + HTML5 Canvas
- **Icons:** Lucide React
- **Font:** Montserrat (Headings) & Inter (Body)

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open the app:**
   Visit [http://localhost:3000](http://localhost:3000)

## Project Structure

- `app/` - Application routes and global styles.
- `components/` - Reusable UI components (Hero, Navbar, ProjectCard, etc.).
- `lib/` - Mock data and utilities.
- `public/` - Static assets (images, SVGs).
